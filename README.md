# Seller Net Sheet

A custom real estate app built by Clave Concept that estimates a home seller's net proceeds at closing.

The seller enters the sale price, mortgage payoff, commission, and closing costs, and the app calculates an estimated net amount the seller walks away with. Commission fields are decoupled to reflect the 2024 NAR settlement, with North Carolina cost defaults built in.

## Features
- Sale price and mortgage payoff inputs
- Decoupled buyer and seller agent commissions
- Editable closing cost line items with NC defaults
- Estimated seller net proceeds, calculated live
- Works on phones, tablets, and desktops on current browsers
- Property photo upload and address-based PDF filename

## Disclaimer
This tool provides estimates for demonstration and planning purposes only. It is not a guarantee of final closing figures. Actual costs vary by transaction, lender, and jurisdiction.
